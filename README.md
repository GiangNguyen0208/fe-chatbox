#  Thực hiện các bước sau để chạy project:
## Install thư viện để thực hiện run project.
```
npm install
```

## Chạy firebase local:
* Trỏ đến folder emulators:
```
cd ./src/emulators
```
* Thực hiện chạy emulator local
```
firebase emulators:start
```


## Cuối cùng chạy project:
```
npm start
```
